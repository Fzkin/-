# 淘宝联盟
通过淘宝联盟淘链接api，回复优惠券信息以及返利信息
使用python web框架，提取xml文件返回消息类型，使用mysql存储用户和订单信息，但推广层面难以突破，项目流产。

公众号介绍文字为
购物，书籍，影视资源为一体的搜索订阅号，还有更多功能等你来反馈添加，默认回复某宝链接，返回优惠券以及返利信息；查找书籍请输入书， 影视资源请输入（影视名称） PS：不要忘加括号亲~
